Bootup computer and F2/Fn+F2 to get to setup.
Enable Secure Boot, and if possible enable Legacy boot (instead of UEFI 
boot)

Or with some computers, just Boot up + F12/Fn+F12 and select the USB 
stick / USB dvd as boot device.

Install Xubuntu (with or without safe graphics)

English

Finnish keyboard layout (or whatever you have/want)
